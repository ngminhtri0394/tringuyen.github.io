Publications

SHAFT: Symmetry-aware Hierarchical Architecture for Flow-based Traversal

T. M. Nguyen, et al.

2024

A novel generative model employing a hierarchical exploration strategy to efficiently exploit the symmetry of the materials space to generate crystal structures given desired properties. The method demonstrates significant advances in crystal structure generation with applications in materials discovery.

Code

Counterfactual Explanation with Multi-Agent Reinforcement Learning for Drug Target Prediction

T. M. Nguyen, T. P. Quinn, T. Nguyen, T. Tran

arXiv preprint arXiv:2103.12983, 2021

A multi-agent reinforcement learning framework (MACDA) to generate counterfactual explanations for drug-protein complex binding affinity prediction. Provides human-interpretable counterfactual instances while optimizing both input drug and target simultaneously for explainable AI in drug discovery.

PDF Code

GEFA: Early Fusion Approach in Drug-Target Affinity Prediction

T. M. Nguyen, et al.

2020

A novel graph-in-graph neural network with attention mechanism addressing changes in target representation due to drug binding effects. GEFA models drugs as graphs of atoms serving as nodes in larger residue-drug complex graphs, creating an expressive deep nested graph neural network for drug-target interaction prediction.

Code

Deep Saliency Detection in Human Crowd Images

T. M. Nguyen, et al.

2019

Saliency detection in human crowd images across different density levels using attention mechanisms. The work addresses the challenging problem of identifying salient regions in crowded scenes through deep learning approaches with attention-based architectures.

Code

RGB-D Super Resolution

T. M. Nguyen, et al.

2023

Super resolution techniques applied to RGB-D images, enhancing both color and depth information simultaneously. The work focuses on improving the quality and resolution of depth-aware image data for computer vision applications.
